# 07 - Documentación Técnica

## Objetivo

Aprender a documentar análisis técnicos, soluciones PL/SQL, decisiones de arquitectura, pruebas y troubleshooting.

La documentación es parte del trabajo de un Arquitecto Oracle Senior.

---

# Temas

- Documentación de Historias de Usuario
- Documentación de Stored Procedures
- Documentación de Vistas
- Documentación de Packages
- Documentación de Performance
- Documentación de Incidentes
- Documentación de Pruebas
- Lecciones Aprendidas

---

# Plantilla Base

Cada documento debe incluir:

- Resumen Funcional
- Análisis Técnico
- Riesgos
- Dudas Funcionales
- Dudas Técnicas
- Solución Propuesta
- Alternativas
- Impacto
- Plan de Pruebas
- Validaciones SQL

---

# Objetivo Final

Poder explicar técnicamente una solución para que otro desarrollador, analista, QA o arquitecto pueda entenderla, mantenerla y validarla.
