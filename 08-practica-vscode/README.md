# Práctica en VS Code

## Objetivo

Practicar SQL y PL/SQL todos los días para memorizar estructuras, sintaxis, patrones y buenas prácticas.

## Método

Cada práctica debe tener:

- consigna
- script base
- solución
- explicación
- errores comunes
- mejora senior

## Temas iniciales

1. Bloques anónimos PL/SQL
2. Variables
3. IF / ELSE
4. LOOP / WHILE / FOR
5. Cursores
6. Procedures
7. Functions
8. Packages
9. Exceptions
10. Bulk Collect / FORALL
