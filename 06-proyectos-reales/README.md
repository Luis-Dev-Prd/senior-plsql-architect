# 06 - Proyectos Reales

## Objetivo

Documentar casos reales, análisis técnicos, troubleshooting, diseños, decisiones arquitectónicas y lecciones aprendidas.

Este directorio representa experiencia práctica aplicada.

---

# Principios

No almacenar:

- código productivo
- datos sensibles
- información confidencial

Sí almacenar:

- análisis
- arquitectura
- patrones
- troubleshooting
- aprendizajes

# Checklist Senior

Debo poder responder:

- ¿Cuál era el problema?
- ¿Qué alternativas existían?
- ¿Por qué se eligió la solución?
- ¿Qué riesgos se identificaron?
- ¿Qué validaciones se realizaron?
- ¿Qué aprendí del caso?
